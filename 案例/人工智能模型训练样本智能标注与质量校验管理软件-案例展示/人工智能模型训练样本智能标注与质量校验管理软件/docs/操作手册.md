# 人工智能模型训练样本智能标注与质量校验管理软件操作手册

## 1. 登录与角色权限

启动 `python main.py` 后进入登录页。系统内置管理员、标注员、质检员和项目经理账号。管理员可访问全部模块；标注员重点使用样本标注生产工作台、文本标注工作台和返工中心；质检员重点使用质量校验中心、一致性分析、Ground Truth 抽检和返工闭环。

## 2. 数据集驾驶舱

驾驶舱展示样本总数、已标注样本、待质检样本、返工数量、智能预标注采用率、自动质检通过率、人工质检通过率、多人一致性评分、Ground Truth 抽检得分、标签均衡度和健康度评分。点击卡片会弹出联动说明，可用于演示业务入口跳转。

## 3. 数据集项目管理

进入“数据集项目管理”后可查看演示项目，也可点击“创建数据集项目”新增图像目标检测、文本分类、实体识别、问答评分等项目。新增后可继续配置标签体系和导入样本。

## 4. 样本导入与数据体检

本模块支持图片文件夹和文本 CSV 导入。图片导入时会检测损坏文件、尺寸异常、完全重复、近似重复等问题；文本导入时会检测空文本、文本过短、标签缺失和重复文本。导入结果会写入 SQLite 数据库，并在表格中展示具体异常项。

## 5. 标签体系与标注规范

标签体系支持标签名称、编码、颜色、类型、快捷键、必填、互斥、正反例和注意事项。标注规范模块可配置图像边界规则、互斥规则、小目标处理规则、文本主诉求规则和评价理由规则。标注工作台会读取这些内容。

## 6. 样本标注生产工作台

工作台采用五区布局：顶部任务条、左侧样本队列、中央标注画布、右侧标签/对象/质检/返工/规范面板、底部状态栏。用户可以筛选未标注、预标注待确认、质检异常、返工样本、低置信和重复疑似样本。

画布支持缩放、拖动、适应窗口、原始尺寸、矩形框绘制、预标注层显示、质检问题层显示、对象定位和删除。点击右侧标签按钮可切换当前绘制标签，在画布拖拽即可新增标注框。点击“智能预标注”会生成候选框；点击“接受全部预标注”会将候选框转为已确认对象。

点击“重新自检”会执行越界框、过小框、重复框、预标注未处理、空标注等规则检查。问题会写入质量问题表，并在右侧实时质检面板中展示。点击“提交审核”会执行提交前强制检查，有中高风险问题时会阻止提交。

## 7. 文本标注工作台

文本工作台支持文本分类、多标签、实体边界记录、问答样本评分和偏好选择。选中文本片段后点击“添加选中实体”可记录实体起止位置。点击“生成大模型样本评分建议”会自动填入偏好和评价理由，便于展示 LLM 训练样本评价流程。

## 8. 质量校验中心

质检员可执行批量自动质量校验，查看高风险样本队列，人工复核通过或退回返工。退回返工时需要填写质检问题说明和返工要求，系统会生成返工单并更新样本状态。

## 9. 多人一致性分析

本模块模拟同一批样本由不同标注员处理后的 IoU 和标签一致率。点击“执行一致性分析”会生成一致性记录，低于阈值的样本会标记为需要仲裁。

## 10. Ground Truth 抽检

管理员可将样本设为标准答案，填写标准答案 JSON 摘要。点击“混入隐藏质检样本”可演示 Ground Truth 样本混入任务队列；点击“计算抽检得分”可汇总标准答案抽检平均分。

## 11. 返工闭环管理

返工中心展示返工编号、样本、标注员、质检员、问题类型、问题说明、返工要求、截止时间、状态和二次质检结果。可推进“返工中、已提交、通过、仲裁处理”等状态。

## 12. 数据集版本管理

版本管理支持冻结当前版本、查看版本记录、模拟差异对比和回滚。冻结版本时系统会读取当前样本数、质检通过率和划分比例，生成可交付版本记录。

## 13. 训练格式导出中心

导出前可执行交付检查，检查未标注、待质检、返工、健康度和标签均衡问题。系统支持导出 YOLO、COCO JSON、Pascal VOC XML 和文本 JSONL，导出记录会写入数据库。

## 14. 报告生成中心

可生成数据集导入体检报告、自动质量校验报告、多人数一致性报告、Ground Truth 抽检报告、返工处理报告、健康度报告和版本交付报告。报告以 Markdown 文件保存到 docs 目录，可在界面预览。
