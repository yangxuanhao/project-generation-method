# 申请表信息

## 主要功能

人工智能模型训练样本智能标注与质量校验管理软件面向 AI 训练数据生产场景，提供从样本入库、数据体检、标签体系配置、智能预标注、人工标注、实时质量校验、质检复核、返工闭环、多人一致性分析、Ground Truth 抽检、数据集健康度评分、版本冻结到训练格式导出的完整流程。系统支持图像目标检测和文本分类/实体/问答评价两类演示项目，内置施工安全帽检测图片样本和客服问答意图分类文本样本。图像工作台提供样本队列、画布缩放、拖动、矩形框绘制、预标注确认、对象列表定位、质量问题高亮、返工说明和提交前强制检查；文本工作台提供单标签、多标签、实体边界、偏好选择、评分维度和评价理由。质量闭环引擎能够检测越界框、过小框、重复框、未确认预标注、空文本、标签冲突、同义文本不一致等问题，并支持质检员退回返工、标注员二次提交和仲裁处理。系统还支持多人一致性评分、Ground Truth 标准答案库、数据集健康度评分、版本冻结、YOLO/COCO/Pascal VOC/JSONL 导出和 Markdown 质量报告生成，适用于教学实训、答辩演示和小范围本地试用。

## 开发目的

提升 AI 训练样本标注、质检、返工和交付管理效率。

## 技术特点

采用 Python3、PyQt6、SQLite、Pillow 等技术，内置规则质检、IoU 一致性、重复检测、健康度评分和多格式导出能力，界面交互完整，支持本地离线运行。
