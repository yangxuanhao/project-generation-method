# 数据格式说明

## 图片数据

支持 jpg、jpeg、png、bmp。导入后系统会记录文件名、路径、宽、高、状态、风险标签、是否 Ground Truth、是否低置信和是否重复。

## 文本CSV

至少建议包含以下字段：

```csv
id,text,label,reason
T001,我想退款,退款,用户明确提出退款诉求
```

## YOLO导出

每张图片对应一个 txt：

```text
class_id x_center y_center width height
```

坐标均为归一化值。

## COCO导出

生成 `annotations_coco.json`，包含 images、annotations、categories 三个数组。

## Pascal VOC导出

每张图片生成一个 XML 文件，包含 filename、size、object 和 bndbox。

## 文本JSONL导出

每行一个训练样本：

```json
{"id":"T001","text":"我想退款","labels":["退款"],"split":"train"}
```
