# 测试说明

1. 使用 admin/admin123 登录，查看全部菜单是否可访问。
2. 进入驾驶舱，点击刷新健康度评分，确认卡片和标签分布表格刷新。
3. 进入样本标注生产工作台，选择图像样本，拖拽新增矩形框，确认对象列表新增。
4. 点击智能预标注，确认画布出现橙色虚线候选框。
5. 点击重新自检，确认右侧实时质检列表出现问题。
6. 进入质量校验中心，点击批量执行自动质量校验，确认问题列表刷新。
7. 选择样本并退回返工，进入返工闭环管理确认返工单生成。
8. 进入一致性分析，点击执行一致性分析，确认生成 IoU 和标签一致率记录。
9. 进入 Ground Truth 抽检，将样本设为标准答案，确认标准答案库刷新。
10. 进入版本管理，冻结当前版本，确认版本记录新增。
11. 进入导出中心，执行交付检查并导出 YOLO/COCO/VOC/JSONL。
12. 进入报告中心，生成健康度报告并预览内容。
