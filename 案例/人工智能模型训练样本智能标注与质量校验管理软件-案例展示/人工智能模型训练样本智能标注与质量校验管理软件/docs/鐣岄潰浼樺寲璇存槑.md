# 界面优化说明

本次版本将页面从“普通管理后台 + 简单表格”优化为更接近真实数据标注生产平台的工作台形态。

## 优化重点

1. **重构主框架**
   - 左侧导航按“总览与项目、生产配置、标注生产、质量闭环、交付与审计”分组。
   - 顶部增加当前业务页面说明、本地数据状态和快速进入生产工作台按钮。
   - 系统名称继续保持为“人工智能模型训练样本智能标注与质量校验管理软件”。

2. **驾驶舱可视化升级**
   - 增加数据集健康度环形仪表。
   - 增加生产流程管线：入库、已标注、待质检、返工、通过、版本。
   - 增加近批次质量趋势线。
   - 增加标签分布柱状图、风险热力图、扣分项和交付建议。
   - 卡片支持点击联动提示，不再只是静态统计。

3. **生产级标注工作台升级**
   - 左侧样本队列改为风险优先队列，支持按低置信、返工、重复、Ground Truth 等维度筛选。
   - 中央画布保留缩放、拖动、框选、预标注层、质检层和标准答案层。
   - 右侧增加“标签、对象、实时质检、提交门禁、AI建议、返工、规范、疑难”工作面板。
   - 提交前门禁会显示预标注确认、质检问题、对象数量、返工处理和疑难说明是否满足要求。
   - AI建议面板会根据当前样本风险、质检问题和预标注状态生成操作建议。

4. **质量校验中心升级**
   - 从普通问题表升级为“质量闭环作战台”。
   - 增加打开问题、质检通过率、一致性评分、返工率等卡片。
   - 增加质检闭环流程：自动扫描、打开问题、待复核、返工中、已通过、仲裁。
   - 增加问题严重度柱状图、问题类型热力图和高风险样本推荐队列。
   - 支持人工复核、退回返工和问题定位说明。

5. **导入与交付页面升级**
   - 导入页面增加导入流水线、体检风险热力图、入库门禁。
   - 导出页面增加交付门禁、健康度仪表、训练/验证/测试划分可视化、导出流程和格式说明卡片。
   - 导出记录继续保留，用于审计留痕。

## 参考的真实系统交互思路

- CVAT 的 Review Mode、Issues、Ground Truth Job 和自动质量估计流程。
- Label Studio 的 Ground Truth 标注、任务一致性和质量评分思路。
- Labelbox 的 model-assisted labeling、benchmark / consensus quality analysis 和 workflow / rework pipeline 思路。

本项目仍然保持本地 PyQt6 + SQLite 可运行，不依赖云端服务。
