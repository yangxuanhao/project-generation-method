# 核心代码说明

项目核心逻辑分为数据库层、算法层、服务层、界面层和控件层。数据库层位于 app/core/database.py，负责 SQLite 表结构初始化、统一查询、写入和日志记录。算法层位于 app/algorithms，包含 bbox 质量检查、IoU、一致性、文本质量、图像哈希、健康度评分和主动复核排序。服务层位于 app/services，负责数据集管理、导入体检、智能预标注、自动质检、多人一致性、返工、导出和报告生成。界面层位于 app/ui，每个菜单一个独立页面文件。控件层位于 app/widgets，其中 image_canvas.py 实现标注画布缩放、拖动、绘框、对象选中和图层显示。
