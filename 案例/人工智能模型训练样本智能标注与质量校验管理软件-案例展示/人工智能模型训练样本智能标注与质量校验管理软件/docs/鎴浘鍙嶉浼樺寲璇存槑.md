# 截图反馈优化说明

本次根据页面截图和使用反馈完成以下优化。

## 1. 窗口大小

主窗口默认尺寸由较大的 `1580×930` 调整为 `1320×820`，同时设置最小尺寸 `1120×720`，适配常见 13/14/15 英寸屏幕。顶部状态栏文案和快捷按钮也已压缩，避免右侧按钮被裁切。

## 2. 重复标题

样本标注生产工作台原先同时显示主窗口标题和页面内部标题。现保留顶部统一标题栏，隐藏工作台内部重复标题，使页面只有一个“样本标注生产工作台”。

## 3. 文字显示不完整

已优化以下组件：

- 队列风险构成热力图：小宽度下自动改为两列，长标签自动换行。
- 提交门禁：检查项高度增加，详情自动换行。
- 工作台工具栏：按钮文字缩短，避免挤压画布和右侧面板。
- 主导航：缩小边距和按钮高度，提高纵向信息密度。

## 5. 更接近真实系统的改动

- 驾驶舱可切换不同演示项目查看健康度、标签分布、风险热力图和生产管线。
- 工作台项目下拉框可切换多个图像项目，不再只有单一图片项目。
- 质量中心能看到跨项目的问题类型，包括图像、文本和表格质量问题。
- 演示数据库内置 Ground Truth、多人一致性、返工单、版本记录和导出记录所需数据。
