路面裂缝人工标注与测量辅助软件——本次最终修复说明

一、页面遮挡修复
1. 主窗口右侧功能区已改为 QScrollArea 滚动容器。
2. 影像任务舱和街景标注舱中的大量输入框、按钮不会再被固定高度裁切。
3. 街景标注舱按钮由横向挤压改为纵向按钮和上一帧/下一帧双按钮行。
4. 任务列表、街景帧列表设置了最大高度，避免列表挤压下方按钮。

二、两点标定修复
1. 比例标定舱新增“关闭两点标定模式”按钮。
2. 两点标定完成后自动切回浏览模式。
3. 取消输入距离时也会清空标定点并回到浏览模式。
4. 避免用户进入其他评定页面后仍继续触发标定点。

三、自动标注增强
1. 自动识别阈值放宽。
2. 暗线连通增强次数增加。
3. 连通域最小长度和形态过滤条件放宽。
4. 自动标注舱默认最多输出 16 条识别结果。
5. 适合街景图中较细、较浅、远距离裂缝的初步标注。

四、街景图替换
1. 删除旧版卡通/模拟街景图。
2. 使用新生成的更真实街景路面裂缝图：
   real_street_crack_view_1.png
   real_street_crack_view_2.png
   real_street_crack_view_3.png
3. 图片位于 assets/streetview/。

五、保留前版功能
1. 图像上参考网格显示和透明度调节。
2. 标注线透明度调节。
3. 最窄、中间、最宽三点宽度估算。
4. 各主要功能舱删除选中标注线。
5. 街景导入接口和 API 接入口占位。
6. 路段位置、坐标、方向、等级和备注记录。
