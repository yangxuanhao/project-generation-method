# 路面裂缝人工标注与测量辅助软件

本项目为 Python3 + PyQt6 桌面端系统，文件夹采用英文名称，软件界面名称保持中文。

## 运行方式

默认账号：

- 用户名：admin
- 密码：admin123

Windows 下双击：

```bat
start.bat
```

或手动运行：

```bat
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python run.py
```

## 本版新增与修复

### 自动标注算法修复说明

本版自动标注舱已重写算法，不再使用上一版容易误检的“全局暗像素连通”方法。现在采用：

- 局部暗线增强，优先识别比周边更暗的细长线；
- 邻域连续性过滤，去掉单个石子、路面颗粒和纹理噪点；
- 连通域几何筛选，过滤块状污渍、边缘脏线和短小碎线；
- 线性支撑度验证，要求拟合折线确实落在暗线区域上；
- 默认“标准”策略，兼顾识别率和误检控制；
- 可手动切换“保守 / 标准 / 增强”三种策略。

同时画布显示也已优化：非选中裂缝不再显示编号标签和宽度文字，避免自动识别后标签大面积重叠。


### 本次再修复

- 右侧功能页已经包入滚动区域，避免影像任务舱和街景标注舱按钮被遮挡或缺损。
- 街景标注舱按钮由横向拥挤排列改为更稳定的纵向/两列按钮排列。
- 两点标定模式新增“关闭两点标定模式”按钮；标定完成或取消后会自动返回浏览模式。
- 自动标注识别力度增强，阈值更宽、连通增强更多、默认最多输出 16 条裂缝候选。
- 已删除旧的卡通街景图，替换为 3 张新生成的更真实街景路面裂缝示例。

### 自动标注算法修复说明

本版自动标注舱已重写算法，不再使用上一版容易误检的“全局暗像素连通”方法。现在采用：

- 局部暗线增强，优先识别比周边更暗的细长线；
- 邻域连续性过滤，去掉单个石子、路面颗粒和纹理噪点；
- 连通域几何筛选，过滤块状污渍、边缘脏线和短小碎线；
- 线性支撑度验证，要求拟合折线确实落在暗线区域上；
- 默认“标准”策略，兼顾识别率和误检控制；
- 可手动切换“保守 / 标准 / 增强”三种策略。

同时画布显示也已优化：非选中裂缝不再显示编号标签和宽度文字，避免自动识别后标签大面积重叠。


1. 参考网格显示在图片上  
   参考网格现在直接覆盖在路面影像上，不再只显示在画布背景中。  
   在“比例标定舱”中可开关图像边缘比例尺、图像参考网格，并可调节参考网格透明度和标注线透明度。

2. 三点宽度估算  
   “宽度卡尺舱”新增“自动估算最窄/中间/最宽三处宽度”。  
   该功能适用于人工标注裂缝，也适用于自动识别裂缝。系统会沿裂缝线多位置采样，然后保留最窄处、中间位置、最宽处三组宽度。

3. 删除标注线功能  
   顶部快捷工具栏、影像任务舱、自动标注舱、宽度卡尺舱、病害诊断舱、网裂圈选舱、复核纠偏舱均可删除选中标注线。  
   自动标注舱额外提供“删除全部自动识别标注线”和“清空疑似候选线”。

4. 实时街景标注舱  
   新增“街景标注舱”，支持：
   - 导入街景图并创建任务；
   - 打开内置街景样例；
   - 上一帧/下一帧切换；
   - 模拟实时街景帧；
   - API 接入口占位配置，暂不接入真实网络请求。

5. 路段信息注明  
   影像任务和街景任务都可以填写：
   - 路段名称；
   - 起止桩号；
   - 位置说明；
   - 经度、纬度；
   - 道路方向；
   - 道路等级；
   - 路段备注。  
   这些信息会写入任务数据，并在导出 JSON、CSV 和报告时保留。


## 建议操作流程

```text
打开普通样例或街景样例 → 填写路段信息 → 标定比例或应用默认比例 → 自动标注舱识别裂缝 → 浏览模式拖动节点修正 → 宽度卡尺舱估算最窄/中间/最宽宽度 → 病害诊断 → 复核纠偏 → 导出报告
```

自动识别属于辅助识别，实际结果建议人工复核和修正后再用于报告。
